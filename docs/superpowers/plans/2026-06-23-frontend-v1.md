# Smart Job Search CRM — Frontend v1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the v1 React frontend for the Smart Job Search CRM: auth (register/login with a route guard), companies, an applications Kanban board with drag-to-update status, interviews, and a dashboard — all consuming the existing Express API.

**Architecture:** Vite + React SPA. Server state via TanStack Query; HTTP via a single axios instance with `withCredentials` and a 401→refresh interceptor. Access token held in memory; the refresh token is an httpOnly cookie the browser sends automatically. An `AuthContext` bootstraps the session on load. Routing via React Router with a `ProtectedRoute` guard. Styling with Tailwind CSS v4. Tests with Vitest + React Testing Library + MSW (the API is mocked at the network layer).

**Tech Stack:** React 18, Vite, react-router-dom v6, @tanstack/react-query v5, axios, Tailwind CSS v4 (`@tailwindcss/vite`), @dnd-kit/core, Vitest, @testing-library/react, @testing-library/user-event, MSW v2.

## Global Constraints

- **API base URL:** from `import.meta.env.VITE_API_URL` (dev default `http://localhost:4000/api`). The backend mounts all routes under `/api`.
- **Credentials:** every API call uses `withCredentials: true` so the httpOnly `refreshToken` cookie (scoped to `/api/auth`) is sent.
- **Auth transport:** access token sent as `Authorization: Bearer <token>`; held in memory only (never localStorage). On reload, the session is restored by calling `/auth/refresh` (cookie-based) to mint a new access token.
- **Error shape from API (verbatim):** `{ "error": { "message": string, "code": string, "details"?: [...] } }`.
- **Application statuses (verbatim, order = Kanban column order):** `Draft, Applied, HR_Screening, Technical_Interview, Final_Interview, Offer, Accepted, Rejected, Withdrawn`.
- **Interview types:** `HR, Technical, Managerial, Final`. **Interview results:** `Pending, Passed, Failed`.
- **Dev server port:** 5173 (Vite default; the backend CORS allows this origin).
- **Module system:** ES modules (Vite default). React function components + hooks only.

## API Contract (consumed)

| Method & path | Body | Success |
|---|---|---|
| `POST /auth/register` | `{email,password,name}` | `201 {user, accessToken}` + sets cookie |
| `POST /auth/login` | `{email,password}` | `200 {user, accessToken}` + sets cookie |
| `POST /auth/refresh` | — (cookie) | `200 {accessToken}` |
| `POST /auth/logout` | — | `204` |
| `GET /auth/me` | — (Bearer) | `200 {user}` |
| `GET /companies?search=` | — | `200 [company]` |
| `POST /companies` | `{name,industry?,website?,location?,size?,notes?}` | `201 company` |
| `PATCH /companies/:id` | partial | `200 company` |
| `DELETE /companies/:id` | — | `204` |
| `GET /applications?status=` | — | `200 [application]` |
| `POST /applications` | `{position, companyId?, status?, ...}` | `201 application` |
| `PATCH /applications/:id` | partial | `200 application` |
| `PATCH /applications/:id/status` | `{status}` | `200 application` |
| `DELETE /applications/:id` | — | `204` |
| `GET /interviews?applicationId=` | — | `200 [interview]` |
| `POST /interviews` | `{applicationId, type, scheduledAt?, interviewer?, notes?, result?}` | `201 interview` |
| `PATCH /interviews/:id` | partial | `200 interview` |
| `DELETE /interviews/:id` | — | `204` |
| `GET /dashboard/summary` | — | `200 {totalApplications, byStatus, upcomingInterviews}` |

`user`: `{id, email, name, createdAt}`. `application`: `{id, companyId, position, status, applicationDate, salaryMin, salaryMax, source, jobDescription, notes, createdAt, updatedAt}`.

## Task 2: API client + in-memory access-token store + 401 refresh interceptor

**Files:**

**Interfaces:**
- Produces:
  - `authToken.js`: `getAccessToken() → string|null`, `setAccessToken(token|null)`, and `onUnauthorized(cb)` / internal `emitUnauthorized()` (a tiny listener registry so the app can react to a hard auth failure).
  - `client.js`: default export `api` (an axios instance, baseURL = `VITE_API_URL`, `withCredentials: true`). Request interceptor attaches `Authorization: Bearer <token>` when present. - [ ] **Step 3: Write `src/api/authToken.js`**

```js
let accessToken = null;
const listeners = new Set();

export function getAccessToken() { return accessToken; }
export function setAccessToken(token) { accessToken = token; }
export function onUnauthorized(cb) { listeners.add(cb); return () => listeners.delete(cb); }
export function emitUnauthorized() { listeners.forEach((cb) => cb()); }
```

- [ ] **Step 4: Write `src/api/client.js`**

```js
import axios from 'axios';
import { getAccessToken, setAccessToken, emitUnauthorized } from './authToken';

const api = axios.create({
  baseURL: import.meta.env.VITE_API_URL || 'http://localhost:4000/api',
  withCredentials: true,
});

api.interceptors.request.use((config) => {
  const token = getAccessToken();
  if (token) config.headers.Authorization = `Bearer ${token}`;
  return config;
});

api.interceptors.response.use(
  (response) => response,
  async (error) => {
    const original = error.config;
    const status = error.response?.status;
    const isRefreshCall = original?.url?.includes('/auth/refresh');
    if (status === 401 && !isRefreshCall && !original._retried) {
      original._retried = true;
      try {
        const { data } = await api.post('/auth/refresh');
        setAccessToken(data.accessToken);
        original.headers.Authorization = `Bearer ${data.accessToken}`;
        return api(original);
      } catch (refreshErr) {
        setAccessToken(null);
        emitUnauthorized();
        return Promise.reject(refreshErr);
      }
    }
    return Promise.reject(error);
  },
);

export default api;
```

- [ ] **Step 4: Write `src/auth/AuthContext.jsx`**

```jsx
import { createContext, useContext, useEffect, useState, useCallback } from 'react';
import * as authApi from '../api/auth';
import { setAccessToken, onUnauthorized } from '../api/authToken';

const AuthContext = createContext(null);

export function AuthProvider({ children }) {
  const [user, setUser] = useState(null);
  const [status, setStatus] = useState('loading');

  useEffect(() => {
    let active = true;
    authApi.fetchMe()
      .then((u) => { if (active) { setUser(u); setStatus('authenticated'); } })
      .catch(() => { if (active) { setUser(null); setStatus('anonymous'); } });
    const off = onUnauthorized(() => { setUser(null); setStatus('anonymous'); });
    return () => { active = false; off(); };
  }, []);

  const login = useCallback(async (creds) => {
    const { user: u, accessToken } = await authApi.login(creds);
    setAccessToken(accessToken);
    setUser(u); setStatus('authenticated');
    return u;
  }, []);

  const register = useCallback(async (body) => {
    const { user: u, accessToken } = await authApi.register(body);
    setAccessToken(accessToken);
    setUser(u); setStatus('authenticated');
    return u;
  }, []);

  const logout = useCallback(async () => {
    try { await authApi.logout(); } finally {
      setAccessToken(null); setUser(null); setStatus('anonymous');
    }
  }, []);

  return (
    <AuthContext.Provider value={{ user, status, login, register, logout }}>
      {children}
    </AuthContext.Provider>
  );
}

export function useAuth() {
  const ctx = useContext(AuthContext);
  if (!ctx) throw new Error('useAuth must be used within AuthProvider');
  return ctx;
}
```

```jsx
import { useState } from 'react';
import { DndContext, useDraggable, useDroppable } from '@dnd-kit/core';
import { useQuery, useMutation, useQueryClient } from '@tanstack/react-query';
import { listApplications, createApplication, updateStatus } from '../api/applications';

export const STATUSES = [
  'Draft', 'Applied', 'HR_Screening', 'Technical_Interview',
  'Final_Interview', 'Offer', 'Accepted', 'Rejected', 'Withdrawn',
];

// Pure, unit-testable drop mapping. overId is the target column (a status).
export function applyDrop({ activeId, overId }, doUpdate) {
  if (!activeId || !overId) return;
  if (!STATUSES.includes(overId)) return;
  return doUpdate(activeId, overId);
}

function Card({ app }) {
  const { attributes, listeners, setNodeRef, transform } = useDraggable({ id: app.id });
  const style = transform ? { transform: `translate(${transform.x}px, ${transform.y}px)` } : undefined;
  return (
    <div ref={setNodeRef} style={style} {...listeners} {...attributes}
      className="border rounded bg-white p-2 mb-2 shadow-sm cursor-grab">
      {app.position}
    </div>
  );
}

function Column({ status, apps }) {
  const { setNodeRef, isOver } = useDroppable({ id: status });
  return (
    <div ref={setNodeRef}
      className={`w-56 shrink-0 rounded p-2 ${isOver ? 'bg-blue-50' : 'bg-gray-50'}`}>
      <h2 className="text-sm font-semibold mb-2">{status}</h2>
      {apps.map((a) => <Card key={a.id} app={a} />)}
    </div>
  );
}

export default function Applications() {
  const qc = useQueryClient();
  const [position, setPosition] = useState('');
  const { data: apps = [] } = useQuery({ queryKey: ['applications'], queryFn: listApplications });

  const create = useMutation({
    mutationFn: createApplication,
    onSuccess: () => { setPosition(''); qc.invalidateQueries({ queryKey: ['applications'] }); },
  });

  const move = useMutation({
    mutationFn: ({ id, status }) => updateStatus(id, status),
    onMutate: async ({ id, status }) => {
      await qc.cancelQueries({ queryKey: ['applications'] });
      const prev = qc.getQueryData(['applications']);
      qc.setQueryData(['applications'], (old = []) =>
        old.map((a) => (a.id === id ? { ...a, status } : a)));
      return { prev };
    },
    onError: (_e, _v, ctx) => qc.setQueryData(['applications'], ctx?.prev),
    onSettled: () => qc.invalidateQueries({ queryKey: ['applications'] }),
  });

  function onDragEnd(event) {
    const overId = event.over?.id;
    const activeId = event.active?.id;
    applyDrop({ activeId, overId }, (id, status) => move.mutate({ id, status }));
  }

  return (
    <div>
      <h1 className="text-2xl font-semibold mb-4">Applications</h1>
      <form className="flex gap-2 mb-4" onSubmit={(e) => { e.preventDefault(); if (position.trim()) create.mutate({ position }); }}>
        <input className="border rounded px-3 py-2 flex-1" placeholder="Position title"
          value={position} onChange={(e) => setPosition(e.target.value)} />
        <button className="bg-black text-white rounded px-4" type="submit">Add application</button>
      </form>
      <DndContext onDragEnd={onDragEnd}>
        <div className="flex gap-3 overflow-x-auto pb-4">
          {STATUSES.map((s) => (
            <Column key={s} status={s} apps={apps.filter((a) => a.status === s)} />
          ))}
        </div>
      </DndContext>
    </div>
  );
}
```

import { useQuery } from '@tanstack/react-query';
import { fetchSummary } from '../api/dashboard';

export default function Dashboard() {

  if (isLoading || !data) return <p>Loading…</p>;

  return (
    <div>
      <h1 className="text-2xl font-semibold mb-4">Dashboard</h1>
      <div className="grid grid-cols-1 md:grid-cols-3 gap-4">
        <div className="border rounded p-4">

          <div className="text-3xl font-bold">{data.totalApplications}</div>
        </div>
        <div className="border rounded p-4">
          <div className="text-sm text-gray-500 mb-1">By status</div>
          <ul className="text-sm">
            {Object.entries(data.byStatus).map(([s, n]) => (
              <li key={s} className="flex justify-between"><span>{s}</span><span>{n}</span></li>
            ))}
          </ul>
        </div>
        <div className="border rounded p-4">
          <div className="text-sm text-gray-500 mb-1">Upcoming interviews</div>
          <ul className="text-sm">
            {data.upcomingInterviews.map((i) => (
              <li key={i.id}>{i.type}{i.scheduledAt ? ` · ${new Date(i.scheduledAt).toLocaleDateString()}` : ''}</li>
            ))}
            {data.upcomingInterviews.length === 0 && <li className="text-gray-400">None</li>}
          </ul>
        </div>
      </div>
    </div>
  );
}
