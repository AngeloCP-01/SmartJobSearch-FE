## V3-10 ‚Äî Editor v6: Image text-wrapping, drag positioning & free placement ‚òë (2026-07-02, merged & pushed to `main`)
**Frontend-only** (BE side of a FE+BE feature carries spec/plan only). **Delivers the long-deferred "v5: floating behind/in-front-of-text signature overlay."** Three sub-batches on one branch: (a) **Text wrapping** ‚Äî the image node became **inline** (was block); a `wrap` attribute drives 5 modes: In line, Break, Wrap (float left/right, text flows around via CSS `float` + `shape-outside`), **In front of text** and **Behind text** (absolutely positioned, free-drag anywhere via `offsetX/offsetY`). A **load-time migration** (`imageContentMigration.js`) wraps legacy top-level/nested block-image nodes in paragraphs so existing docs still open under the inline schema (recursive ‚Äî covers images in table cells/list items). `.editor-sheet { isolation: isolate }` fixes behind-text images (negative z-index was sinking behind the sheet background). (b) **Drag positioning** ‚Äî removed the Align (L/C/R) buttons; positioning is drag-driven. A NodeView "reposition" drag moves the image node to the drop point (`posAtCoords`) with a **drop caret**; Wrap snaps to the nearer side; Break centered by default. (c) **Free-placement relabel** ‚Äî popup modes renamed/reordered to **In line ¬∑ Over text ¬∑ Behind text ¬∑ Wrap around ¬∑ Break**, where "Over text" is the clear drag-anywhere mode. Built subagent-driven across the three batches + final **Opus** reviews per batch. Notable review catches (all fixed + re-verified): inline-node NodeSelection collapse after attribute edits (a shared reselect helper restores it); the shallow‚Üírecursive migration gap (legacy images in tables/lists were being dropped); a click-not-drag committing a (0,0) reposition; and `repositionImageNode` throwing instead of returning null on an off-target drop. Drag is manual/e2e verified; controller browser-verified the wrap reposition (node moved + side flipped + text wrapped). New pure helpers `imageReposition.js` (`sideForX`, `repositionImageNode`) unit-tested. **221 tests** (full suite green, pristine). Spec/plan: BE `docs/superpowers/‚Ä¶2026-07-01-editor-image-text-wrapping‚Ä¶`, `‚Ä¶2026-07-01-editor-image-drag-positioning‚Ä¶`, `‚Ä¶2026-07-02-editor-image-free-placement-labels‚Ä¶`.
